    Quien es Elegible

  • Es Elegible de participar toda persona que tenga 13 años o más.
  • Todo estudiante del I al X de cualquier Universidad o Institución Técnica Educativa que no forme parte de la comisión organizadora y que cumpla con las demás condiciones.

 

Como Registrarme

Tú puedes registrarte como usuario para HuaHCoding aqui

Una vez que tengas tu usuario, para cada Concurso deberá inscribirse como máximo hasta 1 día antes del concurso on-site.

 

Normas de participación

  • El trabajo es individual.
  • Los participantes podrán llevar cualquier material bibliográfico (libros de consulta, programación, etc.) o librerías de código.

 

Como Funciona

Entorno
Ud. puede usar cualquier lenguaje de programación para resolver los problemas. Usando cualquier entorno de desarrollo o editor de texto.

Cómo funciona
El concurso constará de cuatro a seis problemas para resolverlo en un lapso de 2-3 horas.
Cada problema tendrá 1 Input y 1 Output. Después de descargar el Input, El concursante tiene 4 minutos para ejecutar su programa en ese Input y presentar su Output (junto con el código fuente). Si el concursante se toma más de 4 minutos o la respuesta es incorrecta, el intento será registrado como incorrecto. El concursante puede intentar resolver nuevamente el conjunto input/output cuantas veces quiera, pero se necesitará descargar un nuevo archivo Input. El concursante sabrá inmediatamente si su respuesta fue correcta.

Presentando el Código fuente
Ud. debe cargar su código fuente usado para resolver cada conjunto input/output al mismo tiempo que presenta su solución. Por ejemplo, si Ud. genera código automáticamente, presenta el generador. Ud. no puede presentar código fuente después del concurso. El administrador deberá ser capaz de reproducir su salida usando su código fuente.

Puntuación
Por cada problema que Ud. resuelva correctamente, Ud. obtendrá cantidad de puntos indicado para ese problema, Si hay un empate en puntaje la persona con menos tiempo de penalización gana.

Tiempo de Penalización
Es el Tiempo de presentación del último problema correctamente resuelto (el tiempo está medido desde el inicio del concurso) + cuatro minutos por cada presentación incorrecta.

Por ejemplo, si tu has hecho los siguientes intentos:

[17m10s] intento fallido para A
[17m35s] intento fallido para A
[18m15s] intento correcto para A (10 pts)
[29m30s] intento fallido para B
[30m59s] intento correcto para D (30 pts)
[45m11s] intento correcto para C (15 pts)
 

Entonces tu puntuación será 55, con 57m11s de tiempo de penalización (45m11s + 3*4m)
